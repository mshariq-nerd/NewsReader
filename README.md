# How to compile the App

## Requirements
1. Android Studio 
2. Git 

# Download and install:

1. Open cmd/terminal
2. Navigate to your workspace
3. git clone  https://github.com/mshariq-nerd/NewsReader.git
4. Import the project in Android Studio. 

That's all.
